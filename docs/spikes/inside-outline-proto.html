Inside-Outline Prototyp — Baronie Breitenbruck (rot) & Weihenhorst (grün)

Naiv (Stroke zentriert)

Beide Konturen liegen mittig auf der Grenze → an der gemeinsamen Kante überlappen sie, eine Farbe „gewinnt" (Z-Order). Das ist der Ist-Zustand.

A · Clip (innen, auf der Grenze)

Stroke doppelt breit, auf das Polygon-Innere geclippt → sichtbar bleibt die innere Hälfte, exakt auf der Grenze. Zwei Nachbarn: je eine Farblinie auf ihrer Seite, kein Overlap.

B · Inset (negativer Buffer)

Verkleinertes Polygon (Miter-Inset um d) wird gestrokt. Achte auf konkave Ecken / schmale Stellen — dort entstehen Spikes/Selbstüberschneidungen.
