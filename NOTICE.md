# Rechtliche Hinweise und Quellen

## Projektstatus

Avesmaps ist ein offenes, nicht-kommerzielles Fanprojekt von Valentin Schwind.
Das Projekt steht in keiner offiziellen Verbindung zu Ulisses Spiele.

Diese Website und dieses Repository enthalten nicht-offizielle Informationen zu
Das Schwarze Auge und Aventurien. Inhalte koennen offiziellen Publikationen
widersprechen.

## Verwendetes Material

Das Projekt verwendet bearbeitetes offizielles DSA-Kartenmaterial sowie daraus
abgeleitete Daten.

Orts-, Kreuzungs- und Wegdaten wurden aus [avespfade.de](https://avespfade.de/)
teilweise abgeleitet und ergänzt. Laut avespfade wurden einzelne Ortspositionen
teils an das Dereglobus-Projekt angelehnt.

Quellen:

- avespfade.de
- Kartenmaterial:
  http://www.orkenspalter.de/filebase/index.php/File/46-Aventurienkarte-aus-Regionalplänen/

## Wichtiger Lizenzhinweis für GitHub

Dieses Repository gewährt **keine pauschale Open-Source-Lizenz** auf das
enthaltene DSA-bezogene Karten-, Bild- oder Datenmaterial.

Insbesondere stehen bearbeitete Karten, daraus abgeleitete GeoJSON-Dateien und
sonstige DSA-bezogene Assets nur im Rahmen der jeweils einschlägigen Rechte der
Originalquellen und der Ulisses-Fanrichtlinien.

Die Veröffentlichung auf GitHub bedeutet daher **nicht**, dass diese Inhalte
frei unter MIT, CC oder vergleichbaren Lizenzen weiterverwendet werden dürfen.

Das Repository dient der Dokumentation und Veröffentlichung des Fanprojekts
Avesmaps und nicht als allgemeines Materialarchiv für DSA-Inhalte.

Soweit keine ausdrückliche separate Lizenzdatei etwas anderes regelt, verbleiben
die Rechte an selbst erstelltem Code und selbst erstellten Projektinhalten beim
jeweiligen Urheber.
